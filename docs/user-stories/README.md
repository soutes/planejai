## Regras de escrita

- Formato: **Como** usuário, **quero** [ação], **para** [valor]
- Sem Given/When/Then
- Sem story points
- Sem DoD formal
- Citar rota e endpoints relacionados
- Listar explicitamente o que está fora do escopo da US
